title: 解決登入驗證問題
description: 解決無法驗證老客戶的問題
date: "2024-05-03"
---

原本要使用信箱和密碼的舊官網，

在新官網改版後客戶改為只能使用電話登入，

這讓我們在驗證會員時很容易漏掉驗證老客戶的會員資料，

造成在換點時因系統無法比對到會員資料而無法完成兌換。

> 解決辦法: 在程式上加上一層邏輯
> => 如果沒有比對到帳號為電話的會員時，需要再比對電話的欄位，才不會遺漏到老客戶的資料。

```js
    //第二關檢查驗證電話
    if (member === null) {
      let memberVerifiedTel = await this.getMemberByVerifiedTel(tel)
      if (memberVerifiedTel !== null) {
        member = memberVerifiedTel[0]
      }
    }
```
